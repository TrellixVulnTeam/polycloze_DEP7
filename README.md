# polycloze-blacklist

Rule-based classifiers for blacklisting "non-words".

## Usage

```bash
python -m blacklist spa < words.txt > blacklist.txt
```

## Supported languages

- deu (German)
- eng (English)
- fra (French)
- ita (Italian)
- spa (Spanish)

## Other tools

- `blacklist.uncsv` (for extracting first column out of CSV file)
